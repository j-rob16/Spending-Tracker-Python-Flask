It was built under certain conditions, set by the course;
<br>
Project must be built using only:
* HTML / CSS

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

In order to run the product, npm must be installed on your machine in order to download the project's dependencies.

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
